# 配置卡片拖拽排序功能实现计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 使用 vuedraggable 库实现配置卡片的拖拽排序功能，允许用户通过拖拽重新排列配置顺序。

**Architecture:** 在现有的 ConfigCard 网格布局基础上，使用 vuedraggable 包裹 ConfigCard 组件，实现拖拽交互。拖拽完成后调用 store 的 reorder 方法持久化新顺序到 Tauri Store。

**Tech Stack:** Vue 3 + vuedraggable@4.x (基于 Sortable.js) + Tauri Store

---

## 前置知识

- vuedraggable Vue 3 版本使用 `vuedraggable@next` 或 `vuedraggable@4.x`
- 拖拽组件需要使用 `v-model` 绑定数组，组件内部会直接修改数组顺序
- 需要监听 `@end` 事件来触发持久化操作

async function reorderConfigs(newConfigs: EnvConfig[]): Promise<void> {
  // 加密所有配置后保存
  const encryptedConfigs = await Promise.all(newConfigs.map(encryptConfig));
  await saveConfigsToStore(encryptedConfigs);
  // 更新本地状态（保持解密状态）
  configs.value = newConfigs;
}

## 注意事项

- vuedraggable 的 `v-model` 会直接修改绑定的数组，所以需要在 `@end` 事件中持久化
- 使用 `animation` 属性添加平滑的动画效果
- `ghost-class` 用于显示拖拽时的占位样式
- `item-key` 是必须的，用于 Vue 的 diff 算法
